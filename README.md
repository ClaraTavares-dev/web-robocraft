## 🤖 Robocraft  

O **Robocraft** é um projeto que simula a apresentação de um produto fictício de alta tecnologia (o robô **Protheus**), destacando suas vantagens e características de forma organizada e visualmente agradável.  

## ⚙️ Funcionalidades  
- Menu de navegação estilizado com **Flexbox**.  
- Seção principal com **Grid Layout** organizando texto e imagem lado a lado.  
- Destaque visual para descrição do produto, incluindo botão de ação (*call-to-action*).  
- Seção de vantagens estruturada com **Grid**, exibindo ícones, títulos e descrições.  
- Rodapé simples centralizado.  

## 🛠️ Recursos utilizados com CSS  
- **display: grid** → organização do layout principal e vantagens.  
- **grid-template-columns** → distribuição dos elementos em colunas.  
- **gap** → espaçamento entre itens do grid.  
- **display: flex** → usado no menu para alinhar os links de navegação.  
- **flex-wrap** → permite que os itens do menu quebrem a linha em telas menores.
